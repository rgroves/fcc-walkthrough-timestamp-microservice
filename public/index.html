Timestamp Microservice

This is an implementation of the freeCodeCamp Timestamp Microservice project .

This is part of the freeCodeCamp APIs And Microservices Certification curriculum.

Example Calls

  • Example: No Date Passed - /api/timestamp
  • Example: Simple Date Passed - /api/timestamp/2018-01-31
  • Example: Date with Time Passed - /api/timestamp/2018-01-31T14:40
  • Example: Invalid Date Passed - /api/timestamp/abc
